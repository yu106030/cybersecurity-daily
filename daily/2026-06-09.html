1 严重

Check Point VPN 零日漏洞 CVE-2026-50751 — Qilin 勒索团伙在野利用

CVSS 9.3 · CVE-2026-50751 在野利用 · 已发布补丁

Check Point 披露其 Remote Access VPN 和 Mobile Access 产品中存在一个关键身份验证绕过漏洞，已被 Qilin 勒索软件附属组织在零日攻击中利用。攻击者无需有效密码即可建立 VPN 连接，进而入侵企业内网。全球范围内已有数十个组织受影响。

漏洞机制/攻击方式：

  • 类型：身份验证绕过（逻辑流缺陷）
  • 攻击方式：利用已弃用的 IKEv1 密钥交换协议缺陷，无需密码建立 VPN 连接
  • 影响范围：Check Point Remote Access VPN、Mobile Access、AI驱动 Spark 防火墙
  • 后果：进入企业内网后部署 Qilin 勒索软件，使用 Rclone 外泄数据

值得关注的原因：

  • 攻击者在补丁发布前一个月就已利用该漏洞，零日窗口期长
  • 攻击基础设施同时在利用 Palo Alto、Fortinet 和 F5 的 VPN 漏洞
  • 所有使用 IKEv1 的 Check Point VPN 部署应立即升级或切换至 IKEv2
攻击者 IKEv1缺陷 绕过认证 VPN入侵 Qilin勒索
2 严重

Gogs 修补关键零日 RCE 漏洞（v0.14.3）

参数注入 RCE · 2300+实例暴露 已修补 6/7

自托管 Git 服务 Gogs 于 6月7日发布 0.14.3 版本，修补了一个严重的参数注入漏洞。任何已注册用户可通过恶意 Rebase 合并操作执行任意代码，读取所有仓库（含私有）。

受影响组件：

属性详情
受影响版本≤ 0.14.2 / 0.15.0+dev
修复版本0.14.3
暴露实例2300+ 台
攻击前提普通用户（默认开放注册）
3 高危

Meta AI 客服漏洞致 20,000+ Instagram 账户被劫持

20,225 账户 · 7周窗口期 已修复

攻击者利用 Meta AI 驱动的 HTS 账户恢复工具中的逻辑缺陷，在约七周内劫持了 20,225 个 Instagram 账户。该工具未验证提交的邮箱是否属于目标账户，攻击者可为任意账户提交自己控制的邮箱来接收密码重置链接。

4 严重

Ubiquiti UniFi OS Server 三漏洞链实现未认证 Root RCE

CVE-2026-34908/09/10 已修补（5月）

三个漏洞链式利用可实现未认证远程代码执行并获得 root 权限。攻击者可访问摄像头、门禁控制系统及敏感配置数据。受影响版本为 UniFi OS Server 5.0.6 及更早。

5 高危

Shai-Hulud 供应链攻击入侵 PyPI 19 个科学计算包

19个包 · 数十万下载量 持续蔓延

Shai-Hulud 供应链攻击第三波行动，成功入侵 PyPI 上 19 个科学计算相关 Python 包。继 5月攻击 172 个 npm/PyPI 包后，威胁组织持续瞄准开发者工具链。

6. NFCShare Android 恶意软件通过 GitHub 假银行应用传播

NFCShare 新变种通过 GitHub 托管的假银行应用更新传播，利用 IsoDep 接口实时中继 NFC 卡数据，窃取支付卡信息。

7. SoFi 中国香港子公司第三方数据泄露

SoFi 中国香港警告称黑客在第三方供应商数据库中访问了客户信息，26,000 客户个人信息（含香港身份证号）可能泄露。

据 Mandiant 报告，该勒索团伙正通过社会工程冒充 IT 支持人员，甚至派人上门物理访问律所设备窃取数据。

9. C0XMO 僵尸网络利用 DD-WRT 路由器漏洞传播

FortiGuard Labs 发现 Gafgyt 新变种 C0XMO，利用 DD-WRT 路由器缓冲区溢出漏洞（CVE-2021-27137）传播，支持多 CPU 架构并杀死竞争对手恶意软件。
