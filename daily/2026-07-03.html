JetBrains 全系列 IDE 严重漏洞 — 认证绕过 + 账户接管 + 远程代码执行

覆盖 IntelliJ / PyCharm / WebStorm / GoLand / Hub / YouTrack / TeamCity / Kotlin；攻击者可从单点失守控制 CI/CD 与全量开发资产

1 严重

JetBrains 全产品线严重漏洞 — 认证绕过、账户接管、远程代码执行三重打击

CVE-2026-49367 等 全产品线 · 需立即升级

JetBrains 7月2日发布大规模安全更新，修补覆盖整个产品线的多重严重漏洞——这些漏洞组合可实现从零凭据 → 账户接管 → RCE → CI/CD 供应链失守的完整攻击链。受影响组件包括 Hub、YouTrack、IntelliJ、Kotlin、GoLand、TeamCity 等核心开发与 CI/CD 工具。

核心漏洞清单：

组件漏洞类型影响
JetBrains Hub可预测恢复码 → 账户接管任意账户劫持
JetBrains Hub认证信息跨账户附加 → 提权权限提升
JetBrains Hub直接数据库访问 → 认证绕过完整管理员控制
YouTrack直接数据库访问 → 认证绕过工单系统管理员控制
Kotlin构建缓存元数据不安全反序列化构建期 RCE
GoLand不受信任项目配置 → RCE打开恶意项目即触发
IntelliJ IDEACVE-2026-49367 访客用户命令执行IDE 内 RCE
IntelliJ IDEA文件名补全命令注入内容投毒触发 RCE
TeamCityPerforce 连接设置 RCE供应链核心 RCE

值得关注的原因：

  • 覆盖面广：JetBrains IDE 在全球开发者中渗透率极高，所有近期 2024-2026 版本均受影响
  • CI/CD 供应链核心：TeamCity 是大量企业的构建中枢，RCE 意味着所有产物可被污染
  • 认证绕过即可：直接数据库访问的攻击路径完全不需要任何凭据
  • 攻击链可组合：单一组件失守 → 复用凭据 → 另一组件 RCE → 横向至生产
  • 多租户共享部署风险高：跨项目数据泄露 + 构建篡改
攻击者 猜测恢复码/直访 DB Hub/YouTrack 认证绕过 高权限账户接管 TeamCity/IDE RCE 供应链构建投毒
2 严重

Microsoft SharePoint CVE-2026-45659 — CISA KEV 7/4 截止，仅剩 2 天

CVE-2026-45659 CVSS 8.8 · KEV 7/4 截止

CISA 于 7/1 将 Microsoft SharePoint Server 反序列化 RCE 漏洞加入 KEV 目录，FCEB 机构必须在 7/4 前完成修复。最关键的是：微软在 5 月补丁周期发布了修复，但"忘记披露"——安全公告直到 5/21 才发布。CISA 已确认正在被主动利用。

关联攻击者 Storm-2603：与 Warlock 勒索团伙有关联，专门针对本地 SharePoint 漏洞，通过 Velociraptor DFIR 工具混合恶意与可信管理行为。微软 6/22 披露 Storm-2603 与另一个无关攻击组织并行驻留同一网络——使事件响应更困难。

建议：确认 5月 SharePoint 补丁已应用；审计 Site Member 权限；检查 SharePoint 服务器日志中的异常反序列化活动。

3 高危

攻击者先禁用 Defender + Sysmon + WAF，再丢 Mimikatz — 防御压制型杀伤链

Huntress 7/2 披露 UA4fp7R.aspx 隐写 WebShell

Huntress 7/2 披露了一起令人警觉的入侵事件：攻击者系统性禁用 Microsoft Defender、Sysmon、WAF，然后再通过 Mimikatz 提取凭证。这一技术与本周连续披露的 Defender 漏洞（BlueHammer 勒索软件 + RoguePlanet 0day）形成叠加威胁。

禁用顺序：

  • i.bat 禁用 IIS HTTP 日志
  • PowerShell 关闭 Defender 实时监控/行为监控/脚本扫描/样本提交
  • taskkill 终止 Sysmon / Filebeat / Cortex / SentinelOne / Dr.Web
  • Image File Execution Options 冻结监控进程
  • appcmd 卸载 ModSecurity WAF

Steganographic WebShell 关键技术：

  • UA4fp7R.aspx 隐藏在图片文件中
  • 字符串标记 ONEPIECE 用于关联同类 WebShell
  • 同伙 WebShell：03Fl3i.aspx、WRBYTR5750images.aspx、MRBTPS5754images.aspx、jT1Ds.php、RG0eQV6.php

Apple 紧急发布 iOS 26.5.2 — 29+ 漏洞，4 个 WebKit 由 AI 工具发现

5 严重

PTC Windchill CVE-2026-12569 — 持续大规模 WebShell 部署

CVE-2026-12569 CVSS 9.3 · KEV 6/26 收录

PTC Windchill / FlexPLM 反序列化 RCE 漏洞 CVE-2026-12569 持续在野利用——攻击者大规模部署 JSP WebShell 维持持久化访问。Windchill 被广泛用于制造、工程、航空航天、汽车等行业的 IP 与产品设计管理，失陷意味着核心知识产权泄露。

速览区

6. Defender RoguePlanet 0day — 7/8 Patch Tuesday 修复

微软 7/2 透露 RoguePlanet 新 0day（RedSun 系列）将在 7/8 Patch Tuesday 包含修复补丁。在此之前应启用 Tamper Protection 与监控服务停止事件。

7. Adobe ColdFusion 72h 修复窗口今日（7/3）到期

7/1 发布的 6 个 CVSS 10.0 + 5 个 8-9.x ColdFusion 漏洞 72小时修复窗口今日到期。ColdFusion 在 CISA KEV 目录中已有 16 个被在野利用 CVE——历史经验：补丁发布后数天即被利用。

8. Microsoft 漏洞统计：提权类占 42% — 提权成最关键环节

微软 2026 年漏洞统计显示提权 (EoP) 类漏洞占 42%——单类最大。反映现代攻击链现实：初始访问日益商品化，关键提权步骤决定是否从入侵演变为破坏。

9. SimpleHelp CVE-2026-48558 KEV 逾期 — FCEB 强制上报期

7/2 是 SimpleHelp 满分漏洞 CVE-2026-48558 的 FCEB 强制修复截止日。仍未修复的 FCEB 机构自今日起进入 KEV 逾期状态——安全责任将逐级上报至 DHS。

10. KDDI 数据泄露扩展 — 1420 万 ISP 客户凭证外泄

日本电信巨头 KDDI 披露 6/28 攻击事件——1420 万 ISP 客户邮箱 + 密码组合泄露，归因为第三方软件漏洞利用。虽无金融系统影响，但为未来撞库、钓鱼、BEC、账户接管活动提供大量弹药。

11. Tails OS 可被去匿名化 — DirtyClone + pedit COW 组合利用

安全研究人员证实 DirtyClone (CVE-2026-43503) + pedit COW Linux 内核利用可组合去匿名化 Tails OS 用户——Bypasses Tails 关闭时的内存擦除过程。高风险用户应立即更新 Tails 至最新版本。
