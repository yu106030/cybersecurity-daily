IronWorm npm供应链攻击：蠕虫式自我传播的Rust信息窃取器

36个npm包感染 | 发现日期：6月4日 蠕虫传播

安全公司Ox Security发现一起技术复杂的供应链攻击感染了npm仓库中的36个软件包，植入了名为IronWorm的信息窃取恶意软件。该恶意软件使用Rust编写，具备eBPF内核rootkit隐藏能力和蠕虫式自我传播机制。

漏洞机制/攻击方式：

  • 类型：供应链投毒 + 蠕虫传播
  • 入口：被入侵的asteroiddao账户发布恶意包版本
  • 执行：通过npm preinstall脚本自动触发
  • 隐藏：eBPF内核rootkit + Tor网络C2通信
  • 窃取目标：86个环境变量 + 20类凭据（AWS/OpenAI/Anthropic/npm/GitHub/SSH/加密钱包）

值得关注的原因：

  • 蠕虫式传播：窃取凭据后自动发布木马化版本，引爆CI/CD链条
  • Git提交伪装为claude，时间戳篡改至13年前
  • 与Shai Hulud恶意软件存在代码关联，可能是其演进版本
入侵账户 发布恶意包 preinstall触发 rootkit隐藏 窃取凭据 蠕虫传播

Claude Code GitHub Action漏洞：恶意Issue劫持仓库

CVSS v4.0 7.8 | 修复：v1.0.94 供应链风险

Anthropic的Claude Code GitHub Action存在安全缺陷，攻击者仅需发布一个恶意GitHub Issue即可接管使用该Action的公共仓库。

攻击链：

  • 绕过验证：注册GitHub App获取[bot]身份绕过写权限检查
  • 提示注入：Issue中构造隐藏指令读取/proc/self/environ
  • 凭证窃取：窃取OIDC令牌换取Claude App写入权限
  • 供应链投毒：修改官方Action代码感染下游项目
