Adobe ColdFusion APSB26-68 — 11 个严重漏洞，6 个 CVSS 10.0 满分未认证 RCE

APSB26-68 CVSS 10.0 × 6 · 72小时修复

7 月 1 日 Adobe 发布 ColdFusion 2025 和 2023 的紧急安全更新（APSB26-68），一次性修补 11 个严重漏洞，其中 6 个达到 CVSS 10.0 满分。所有 6 个 10.0 漏洞均允许未认证远程攻击者在 ColdFusion 服务器上执行任意代码。Adobe 已将其列为最高优先级，建议 72 小时内完成修复。

Chrome 151 — 史上单版本最多：382 个漏洞，15 个严重 RCE

Chrome 151 382 漏洞 / 15 RCE

7 月 1 日 Google 发布 Chrome 151，单次修补 382 个安全漏洞——这是 Chrome 历史上单版本漏洞数量最多的一次。其中 15 个被评定为严重级别，可导致远程代码执行。

建议：验证全网终端 Chrome 151 部署；Chromium 内核浏览器（Edge / Opera / Brave）需同步更新。

3 严重

Microsoft Defender "RoguePlanet" 0-Day — 一周内第二个 Defender 0日

Defender 0-Day 无补丁 · 微软开发中

7 月 1 日微软确认存在新的 Defender 严重零日漏洞，代号 "RoguePlanet"，目前无补丁可用。这是一周内披露的第二个 Microsoft Defender 零日漏洞——仅一天前 CISA 才刚确认 BlueHammer（CVE-2026-33825）被勒索软件团伙在野利用。

事件时间线：

  • 6 月初：BlueHammer CVE-2026-33825
  • 6/30：RoguePlanet 第二变种 CVE-2026-50656
  • 7/1：RoguePlanet 新 0day — 无补丁
  • 关联：RedSun / UnDefend 同研究团队系列

讽刺性：用于保护系统的杀毒软件反而成为提权途径。Defender 以 SYSTEM 权限运行——任何漏洞即意味着完整系统接管。

4 严重

CISA KEV 截止日：SimpleHelp CVE-2026-48558 今日（7/2）强制修复

CVE-2026-48558 CVSS 10.0 · FCEB 7/2 截止

昨日头条 SimpleHelp CVE-2026-48558 进入 CISA KEV 目录第 4 天——今日（7 月 2 日）是联邦机构（FCEB）的强制修复截止日。该 RMM 远程监控管理软件的认证绕过漏洞 CVSS 满分，攻击者已利用 TaskWeaver + Djinn Stealer 双料武器对 MSP 客户终端发起大规模攻击。

武器特征：

  • TaskWeaver：Node.js 重混淆加载器
  • C2：滥用微软合法 Dev Tunnels（a.dev-tunnels.com）
  • Djinn Stealer：跨 Win/macOS/Linux，窃取 AI 助手凭据

链式反应：单一 RMM 沦陷 → 数千 MSP 客户终端失陷。

5 高危

WinRAR 7.23 紧急修复 — RAR5 格式 25 年老漏洞变种 RCE

EUVD-2026-40869 RAR5 恢复卷 RCE

WinRAR 发布 7.23 版本修复一个长期潜伏的 RCE 漏洞 EUVD-2026-40869，该漏洞位于 RAR5 格式的恢复卷（Recovery Volume）处理器。这是 CVE-2023-40477 的变种——2023 年原始漏洞仅修补了 RAR3 格式，RAR5 格式的相同缺陷一直遗留至今。

触发条件：

  • 打开恶意压缩包
  • 访问触发下载的恶意网页
  • 影响：全网 Windows 桌面终端

特殊问题：WinRAR 无自动更新机制，不支持 Group Policy 部署。趋势科技报告显示旧 WinRAR 漏洞仍被积极利用正是因为这一补丁部署空白。

建议：手动升级至 7.23，或考虑迁移 7-Zip（支持企业部署）。

速览区

6. libssh2 CVE-2026-55200 — CVSS 9.2 整数溢出 PoC 已公开

SSH 数据包长度字段未校验导致整数溢出。攻击者通过搭建恶意 SSH 服务器在握手阶段利用客户端 libssh2 RCE。影响几乎所有 Linux 服务器（ssh / curl / git 等基础工具）。

7. Cisco Unified CM CVE-2026-20230 — SSRF 在野利用已确认

未认证 HTTP 请求触发 WebDialer 服务 SSRF，可任意文件写入 + 提权至 root。Cisco 6/3 发布补丁，但 PoC 公开后攻击激增。

8. Splunk Secure Gateway CVE-2026-20251 — 反序列化 RCE PoC 已公开

Splunk Secure Gateway（SSG）反序列化 RCE 漏洞，公开 PoC 显著提高利用风险。建议立即应用 Splunk 补丁，限制 SSG 访问至可信网络。

9. Rancher CVE-2026-41053 — GitHub 认证缓存绕过 CVSS 8.8

Rancher Kubernetes 管理平台 GitHub 认证提供程序认证缓存错误，授予任何已登录用户访问权限——完整认证绕过。影响 Rancher 2.13 < 2.13.6 与 2.14 < 2.14.2。

10. Apache Tomcat — 认证绕过 + 安全约束绕过

Tomcat 披露多个认证与安全约束绕过漏洞。Tomcat 是部署最广泛的 Java 应用服务器之一，认证绕过可能暴露内部应用。建议立即更新。

11. Apache ActiveMQ — 6 个新 CVE（7.5-8.1 分）

ActiveMQ 消息中间件披露 6 个输入验证、授权、内存处理漏洞。建议立即更新。
